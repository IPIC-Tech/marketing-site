# Marketing Site

This package will provide the public marketing site, landing pages, and campaign management for the InstaPass platform.

## Getting Started
- Install dependencies: `npm install`
- Run tests: `npm test`

## Structure
- `src/` - main code
- `src/types.ts` - type definitions
- `src/index.ts` - marketing logic
- `src/index.test.ts` - tests

## Features
- Landing pages
- Campaign management
- Analytics

## Setup
- Add marketing content
- Integrate with platform services
